#### 6，约束

理解什么是约束

用来固定草图几何形状的规则。
- 分类：
  - **几何约束**：控制几何关系（如平行、垂直、重合、相等等）。
  - **尺寸约束**：带数值的约束（如长度=50mm，直径=20mm）。
- 特点：
  - 几何约束决定“形状”，尺寸约束决定“大小”。
  - 草图完全约束后，所有线条会变黑，无法随意拖动。
- 小结：**约束 = 形状 + 大小 + 定位**，三者齐全才能完全锁定草图。



举例子：

1，一个矩形 加上长和宽的尺寸  就加上了尺寸约束 无法再改变长宽大小

2，一个矩形  将某个点和坐标点加上重合约束  这个矩形就无法再移动位置  

3，一个矩形 本身就具有平行约束  即两套边平行

4， 两个矩形  调整一个矩形1  **用相等约束让矩形2具有和矩形1 相同的约束**

#### 8，衍生

把已经创建的零件放进inventor



衍生是 **Inventor 自己的关联性功能**，不是单纯复制。

它允许你在一个零件里 **引用另一个零件/装配体的几何、基准、参数**，并保持**参数关联**。

意义：

1. 你可以选择要带进来的东西（实体、曲面、草图、工作平面、参数…）。
2. 如果源零件改了，衍生出来的零件会自动更新。
3. 常见于 **主模型法(Top-down design)**：先画一个“母件”，再衍生出多个子零件。













## 二，部件模块与其他

###  强相关（推荐看）

1. **Inventor部件模块_放置零件模块.mp4**
   - 涉及零件的放置与约束 → 后续你要处理装配体时，理解“零件位置参数”很关键。
   - 对标注来说，坐标/相对位置是必不可少的。
2. **Inventor部件模块_结构件加速器+衍生.mp4**
   - 里面有衍生功能 → 我之前解释过，衍生能把母体数据传给子零件。
   - 这和你想要“抽取几何数据”有一定关系，特别是分段轴类零件时可能用得到。
3. **Inventor部件模块_设计加速器.mp4**
   - 通常包含标准件、轴、孔、键槽等自动化生成。
   - 对你的“轴类零件自动标注”非常相关，因为这些特征是智能标注的重点。
4. **Inventor部件模块_约束和对称(参数+标注).mp4**
   - 这个直接涉及“约束”和“参数化”，是理解“尺寸/标注背后的数据”的核心。
   - 建议认真看，对你以后导出 JSON 有帮助。

------

### 📌 次相关（可跳着看）

- **BOM表.mp4**
  - 偏向装配件清单管理，对智能标注不是直接必要，但如果后续要做“批量标注/参数统计”，有点用。
- **标准件库+标准件生成.mp4**
  - 跟轴承、螺栓之类相关，跟“标注”不是直接关联，但如果你想做“自动识别孔径+配螺栓标注”，可以作为扩展。
- **参数变量.mp4**
  - 跟参数化设计相关，间接重要。你要搞明白“标注的数字是从哪来的”，参数化就是源头。

------

### 📌 弱相关（可以先不看）

- **工程塑型单元_FDU.mp4**
  - 偏模具/塑料件方向，和轴类零件自动标注关系不大。
- **手势命令及快捷键.mp4**
  - 主要是操作效率提升，对标注逻辑帮助有限。
- **Vault综合.mp4**
  - 是PDM/数据管理系统的内容，和标注任务关系不大。
