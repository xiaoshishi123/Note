# **PART 01｜企业内部视频网站项目**

# （全年持续推进）

------

# **P3｜项目概述：建设目标与主线**

1，项目背景

- 公司内部培训视频不断增多，缺乏统一管理平台
- 需建设可上传、管理、播放、统计的一体化系统
- 提供给技术中心、业务部门使用

2，全年推进主线

- **一期（1–3 月）**：上线基础功能（登录/播放/分类/前端样式）
- **二期（9–11 月）**：建设部门统计、学习任务汇总
- **三期（预研）**：平台综合化（用户会话、学习链路）

# **PART 02｜结构插件移植项目（4—8 月）**

------

# **P11｜项目背景与目标**

1，背景

- 原有结构工具插件分散、依赖旧版 CAD 环境
- 多个工具条功能失效或难以维护
- 需重新移植并统一结构工具体系

2，目标

- 工具条界面重构
- 旧版功能逻辑复刻 + 优化
- 基础代码模块化、规范化
- 积累部门可持续使用的工具库
